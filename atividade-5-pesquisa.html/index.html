Elementos HTML ""

O elemento HTML "" define um texto em negrito, sem nenhuma importância extra.

Use "" quando quiser destacar algo visualmente, mas que não precisa ter uma ênfase semântica especial. Por exemplo, nomes técnicos, termos-chave ou palavras importantes, mas sem indicar urgência ou importância.
