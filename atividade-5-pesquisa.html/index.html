Elemento HTML <b>

O elemento HTML <b> define um texto em negrito, sem nenhuma importância extra.

Use <b> quando quiser destacar algo visualmente, mas que não precisa ter uma ênfase semântica especial. Por exemplo, nomes técnicos, termos-chave ou palavras importantes, mas sem indicar urgência ou importância.

Exemplo: 

                Este é um texto normal, e este é  texto em negrito .
